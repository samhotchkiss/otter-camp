Usage Guidelines

Backgrounds

Use Warm Cream (#FAF8F5) as the default background. Foam (#F5F2ED) for cards and elevated surfaces. Never use pure white â€” keep it warm.

Text Hierarchy

Deep Earth (#2D2A26) for headings. Stone (#6B6560) for body. Pebble (#A69582) for muted/meta. Bark (#8B7355) for labels.

Interactive Elements

Otter Brown (#5C4A3D) for primary buttons/CTAs. River Blue (#4A6D7C) for links. Sunset Orange (#C87941) for attention-grabbing CTAs.

Status Colors

Forest Green (#5A7A5C) for success. Sunset Orange (#C87941) for warnings. Campfire (#B85C38) for errors. Tide Pool (#6B8F9C) for info.
