Crankfeed Design Decisions

  • Ambient, not attention-seeking: No notifications, user pulls when curious
  • Time-grouped: Easy to scan "what happened in the last hour?"
  • Collapsible history: Don't overwhelm with old data
  • Agent avatars: Quick visual recognition
  • Project badges: Context without leaving feed
  • Expandable details: Commit messages inline
  • Live indicator: Confidence that feed is real-time
  • Filters: Slice by type or project
